## 技术实现细节

### API路由 (`app/api/search/route.ts`)
- 接收前端搜索请求
- 代理到盘搜API: http://124.220.76.89:8080/api/search
- 支持参数: q(关键词), page(页码), size(每页数量), type(网盘类型)
- 缓存机制减少重复请求
- 错误处理和超时控制

- 移动端菜单支持

## 盘搜API对接

### 接口地址
- 基础URL: http://124.220.76.89:8080/api
- 搜索接口: /search?q=关键词&page=页码&size=数量

### 响应格式
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "total": 3445,
    "merged_by_type": {
      "magnet": [...],
      "baidu": [...],
      "quark": [...],
      "aliyun": [...]
    }
  }
}
```

### 支持的网盘类型
- magnet (磁力链接)
- baidu (百度网盘)
- quark (夸克网盘)
- aliyun (阿里云盘)
- 115 (115网盘)
- pikpak (PikPak)
- tianyi (天翼云盘)
